# Violet Chat

Готовый статический проект фейкового мессенджера для iPhone / GitHub Pages.

## Что уже есть
- чат в стиле iMessage с фиолетовым акцентом
- отправка сообщений и автоответы
- постоянное время `12:47`
- кликабельный аватар
- открытие фото во весь экран
- zoom жестами и двойным тапом
- PWA режим для запуска с экрана Домой

## Как заменить фото
В папке `assets`:
- `avatar.jpg` — маленький круглый аватар
- `profile.jpg` — большое фото

Лучше использовать латиницу, без пробелов.

## Как опубликовать
1. Загрузи все файлы в репозиторий GitHub.
2. Включи GitHub Pages: `Settings -> Pages -> Deploy from a branch -> main -> /(root)`.
3. Открой сайт в Safari.
4. Добавь на экран Домой.

## Если сайт не обновляется
- удали старую иконку с экрана Домой
- открой сайт заново в Safari
- снова добавь на экран Домой
