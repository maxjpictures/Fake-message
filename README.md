# Violet Chat — final build

Готовый статический проект для GitHub Pages и запуска с экрана Домой на iPhone.

## Что внутри
- реалистичная анимация появления сообщений
- фиолетовые исходящие сообщения
- постоянное время 12:47
- кликабельный аватар
- полноэкранное фото с pinch/double tap/drag
- PWA manifest

## Файлы фото
- `assets/avatar.png` — маленький аватар
- `assets/mazeratti.jpg` — большое фото

## Публикация
1. Загрузить все файлы в корень репозитория.
2. Включить GitHub Pages из ветки `main`, папка `/ (root)`.
3. Открыть сайт в Safari и добавить на экран Домой.
