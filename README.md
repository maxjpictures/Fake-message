# Violet Chat — фейковый мессенджер в стиле iPhone

Готовый статический проект для GitHub Pages.

## Что внутри

- интерфейс, похожий на iMessage, но со своим фиолетовым стилем
- отправка ваших сообщений в пузырях справа
- автоматические ответы из заранее подготовленного массива
- кликабельная круглая аватарка собеседника
- открытие фото во весь экран
- увеличение фото кнопками, колесом мыши, двойным тапом и pinch-жестом на телефоне
- поле ввода с плейсхолдером `сообщение`
- полностью статический сайт без сборки и зависимостей

## Структура

## Как быстро заменить ответы

Откройте `script.js` и измените массив:

```js
const scriptedReplies = [
  'Привет. Я уже здесь ✨',
  'Да, выглядит очень похоже на настоящий чат.',
  'Можно заменить мой аватар на любое твое фото.',
  'На GitHub Pages это будет работать как обычный сайт.',
  'Если хочешь, можно потом добавить еще ветки диалога.'
];
```

## Как заменить аватарку и большое фото

1. Замените файл `assets/avatar.svg` на свою круглую картинку.
2. Замените файл `assets/profile-large.svg` на полное фото.
3. Если хотите использовать JPG или PNG, просто поменяйте пути в `index.html`.

Пример:

```html
<img src="assets/avatar.jpg" alt="Аватар собеседника" class="avatar" />
<img src="assets/profile-large.jpg" alt="Полное фото собеседника" id="zoomImage" class="zoom-image" />
```

## Публикация на GitHub Pages

### Вариант 1 — через сайт GitHub

1. Создайте новый репозиторий.
2. Загрузите в него все файлы проекта.
3. Откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**.
5. Выберите ветку `main` и папку `/ (root)`.
6. Сохраните настройки и дождитесь публикации.

### Вариант 2 — через Git локально

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ-ЛОГИН/fake-iphone-messenger.git
git push -u origin main
```

После этого включите GitHub Pages в настройках репозитория.
